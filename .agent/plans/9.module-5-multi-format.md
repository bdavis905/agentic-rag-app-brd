## Task 2: Replace `extract_text` with Docling-based extraction

**File:** `backend/app/services/ingestion_service.py`

1. Add imports: `asyncio`, `tempfile`, `os`, `DocumentConverter` from docling
2. Add `EXTENSION_TO_FILE_TYPE` reverse map and a singleton `DocumentConverter` instance (expensive to create, reuse)
3. Replace sync `extract_text()` (line 100) with an async version:
   - Fast-path: `.txt`/`.md` → simple UTF-8 decode (no Docling overhead)
   - All other types: write bytes to a temp file → run `converter.convert(path)` in a thread pool executor → `export_to_markdown()` → clean up temp file
4. Update line 39 from `text = extract_text(...)` to `text = await extract_text(...)`

**Key design:**
- Singleton converter avoids repeated model loading
- `asyncio.get_event_loop().run_in_executor(None, ...)` keeps the event loop unblocked during CPU-heavy PDF/image parsing
- `tempfile.NamedTemporaryFile(delete=False)` + explicit `os.unlink` in `finally` handles Windows temp file constraints
