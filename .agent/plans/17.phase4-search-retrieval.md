# Phase 4: Search + Retrieval

**Prerequisite:** Read `.agent/plans/15.convex-migration-progress.md` first for context.
**Depends on:** Phase 3 complete (chunks with embeddings in database)

**Complexity:** ⚠️ Medium

## Goal
Vector search, text search, hybrid search with RRF, optional reranking.

## Reference Files (Python originals)
- `backend/app/services/retrieval_service.py` — vector search + hybrid fusion
- `backend/app/services/rerank_service.py` — Cohere-compatible reranking

## Steps

### 1. Create `convex/search/actions.ts`
- `"use node"` directive

#### vectorSearch action
- Uses `ctx.vectorSearch("chunks", "by_embedding", { vector, limit: 16, filter: q => q.eq("userId", userId) })`
- Takes a query string, embeds it first (reuse embedding.ts), then searches
- Returns document IDs + scores + content

#### textSearch action
- Uses Convex `searchIndex` on chunks table
- `ctx.db.query("chunks").withSearchIndex("search_content", q => q.search("content", query).eq("userId", userId))`
- Returns matching chunks with relevance

#### hybridSearch action
- Runs vectorSearch and textSearch in parallel
- Fuses results with Reciprocal Rank Fusion (RRF)
- RRF formula: `score = sum(1 / (k + rank))` where k = 60
- Port the RRF algorithm from `retrieval_service.py`
- Returns fused results sorted by combined score

#### rerank action (optional)
- Calls Cohere-compatible rerank API via fetch
- Uses `RERANK_API_KEY`, `RERANK_BASE_URL` env vars
- Only runs if env vars are configured
- Reorders results by relevance to query

### 2. Update `convex/_generated/api.ts` stub
- Add search action references to the api object

## Validation
- Search for content known to be in uploaded docs → vector results returned
- Text search returns matching chunks
- Hybrid search returns fused results from both
- If rerank configured: results are reordered

## Key Algorithm: Reciprocal Rank Fusion
```
For each result in vector_results:
  rrf_score[doc_id] += 1 / (k + vector_rank)
For each result in text_results:
  rrf_score[doc_id] += 1 / (k + text_rank)
Sort by rrf_score descending
```
